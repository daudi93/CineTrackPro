# Changelog

All notable changes to CineTrack Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [2.1.0] - 2026-03-17

### Added
- 🎯 **Advanced Object Tracking** - 7 tracking modes including Follow, Orbit, Lead, Trail, Frame, Locked, and Predictive
- 👤 **Operator Behavior Simulation** - Realistic human camera operation with breathing, fatigue, and micro-adjustments
- 🖼️ **Intelligent Framing Rules** - Automatic rule of thirds, headroom control, and horizon stabilization
- 📋 **Shot Management System** - Create, manage, and execute professional camera shots
- 🎥 **Camera Rig Generator** - Generate dolly, steadicam, car rigs, and SnorriCam
- 📊 **Multiple Target Tracking** - Track and blend between multiple subjects
- 🔮 **Motion Prediction** - Anticipate subject movement for smoother tracking
- ⚡ **Quick Shot Presets** - One-click shots (Close-up, Medium, Long, POV, Tracking)

### Fixed
- 🐛 Fixed auto-rotation issue when add-on is installed but not in use
- 🐛 Improved tracking accuracy with object rotation
- 🐛 Fixed physics damping calculations
- 🐛 Corrected shot execution duration handling
- 🐛 Fixed manifest tags for Blender 5.0 compatibility

### Changed
- ⚡ Optimized performance with reduced history calculations
- 📝 Updated documentation with comprehensive user guide
- 🎨 Improved UI layout with collapsible sections
- 🔧 Enhanced error handling and debug information
