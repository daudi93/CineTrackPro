## Keyboard Shortcuts

| Shortcut | Function |
|----------|----------|
| `N` | Toggle sidebar (Camera Motion tab) |
| `Space` | Play/Stop animation |
| `Alt + A` | Bake current shot |
| `Shift + T` | Toggle tracking |
| `Shift + S` | Add quick shot |

## Common Issues Quick Fix

**Camera moves when add-on installed?**
→ Make sure Tracking/Shake are disabled until ready to use

**Not following target?**
→ Check target object is selected and tracking is enabled

**Too jerky/smooth?**
→ Adjust smoothing values in Tracking panel
